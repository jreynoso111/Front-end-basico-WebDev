El modelo de caja

Controla márgenes, bordes y rellenos para que tus elementos ocupen el espacio correcto.

Recuerda

  • margin: espacio externo; padding: espacio interno.
  • border suma grosor a la caja, ajusta box-sizing: border-box para facilitar cálculos.
  • Usa un inspector del navegador para ver el diagrama de caja en vivo.

Práctica

Crea tres tarjetas con diferentes márgenes y rellenos, observa cómo afectan el layout.

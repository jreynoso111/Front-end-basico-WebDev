2. ¿Por qué importa? (Div vs Semántica)

A la izquierda, código "malo" (funciona pero es ilegible). A la derecha, código "profesional".

❌ Código sucio (Div Soup) 
<div id="header">...</div>
<div id="menu">...</div>
<div class="contenido">...</div>
<div id="pie">...</div>
✅ Código Semántico 
<header>...</header>
<nav>...</nav>
<main>...</main>
<footer>...</footer>

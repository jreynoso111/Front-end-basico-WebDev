Funciones y condicionales

Organiza tu lógica en bloques reutilizables y toma decisiones según condiciones.

Conceptos clave

  • Declara funciones con function o como const saludar = () => {}.
  • Condicionales: if, else if, else. Evalúan expresiones booleanas.
  • Retorna valores con return para reutilizarlos en otras partes del código.

Ejercicio

Crea una función puedeConducir(edad) que devuelva mensajes diferentes según la edad y pruébala con varios números.

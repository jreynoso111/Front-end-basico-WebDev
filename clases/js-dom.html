JavaScript: El DOM (Manipulación)

El DOM (Document Object Model) es la herramienta que te permite "agarrar" elementos HTML y cambiarlos sin recargar la página.

Explicación de los Métodos

  • querySelector('#id'): Es como un gancho. Busca en el HTML y "pesca" el primer elemento que coincida.
  • textContent: Permite leer o sobrescribir el texto que vive dentro de la etiqueta.
  • style.propiedad: Modifica el CSS en línea. (Nota: usa CamelCase, ej: backgroundColor en vez de background-color).
  • classList.toggle(): Es un interruptor. Si la clase existe, la quita; si no existe, la pone. Es ideal para menús y modos oscuros.

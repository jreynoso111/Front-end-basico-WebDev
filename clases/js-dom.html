DOM: selecciona y modifica elementos

Conecta JavaScript con el HTML para leer y actualizar la interfaz.

Selección

  • document.querySelector('.clase') devuelve el primer elemento que coincide.
  • document.querySelectorAll('p') devuelve una NodeList iterable.
  • Guarda las referencias en variables para reutilizarlas y mejorar el rendimiento.

Manipulación

Cambia textos con element.textContent, clases con classList.add() y atributos con setAttribute().

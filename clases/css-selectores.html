Sintaxis y selectores

Aprende a apuntar elementos con precisión usando clases, IDs y selectores combinados.

Claves rápidas

  • Prefiere clases para reutilizar estilos, usa ID solo para elementos únicos.
  • Combina selectores: .card p selecciona párrafos dentro de .card; button.primary un botón con clase primary.
  • La especificidad importa: inline > ID > clase > elemento.

Práctica

Estiliza una tarjeta con fondo blanco y texto gris, y agrega un botón azul usando clases.

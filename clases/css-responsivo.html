Diseño responsivo

Adapta tus layouts a distintos tamaños de pantalla con media queries.

Guía rápida

  • Define un breakpoint base (ej. 768px) y ajusta a móviles primero.
  • Usa unidades relativas (rem, %, vw) para tamaños fluidos.
  • Prueba en el inspector con diferentes dispositivos simulados.

Ejercicio

Convierte un layout de tres columnas en una sola columna debajo de 768px usando @media.

Variables y tipos de datos

Aprende a guardar y manipular información en JavaScript.

Conceptos clave

  • Declara con const para valores que no cambian y let para los que sí.
  • Tipos básicos: string, number, boolean, null, undefined.
  • Usa typeof para inspeccionar el tipo de una variable.

Mini práctica

Crea variables con tu nombre, edad y si te gusta el front-end. Imprime una frase combinando los valores en la consola.

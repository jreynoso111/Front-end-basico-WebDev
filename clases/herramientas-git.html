Git básico: versiones de tu código

Registra cambios y colabora con otros desarrolladores.

Comandos esenciales

  • git init, git status, git add ., git commit -m "mensaje".
  • Crear y cambiar ramas: git branch nombre, git checkout nombre o git switch nombre.
  • Subir a remoto: git remote add origin URL y git push -u origin main.

Práctica

Inicializa un repositorio en esta carpeta, haz un commit con tus notas y crea una rama para experimentar.

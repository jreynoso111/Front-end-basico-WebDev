Sintaxis básica y estructura del documento

Conoce las piezas imprescindibles que forman un archivo HTML y cómo organizarlas de forma correcta.

Objetivo

Crear un documento HTML mínimo que el navegador pueda interpretar sin errores, con una jerarquía clara.

Conceptos clave

  • <!DOCTYPE html>: indica que el documento usa HTML5.
  • <html lang="es">: envuelve todo el contenido y define el idioma.
  • <head>: metadatos, título, enlaces a hojas de estilo y scripts.
  • <body>: todo lo que el usuario verá en la página.
Mini práctica: crea un archivo index.html con las etiquetas anteriores y agrega un título y un párrafo dentro del <body>.

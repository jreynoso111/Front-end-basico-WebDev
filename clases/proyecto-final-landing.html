Proyecto final: landing page personal

Integra HTML, CSS y JavaScript construyendo una página de presentación.

Requisitos mínimos

  • Estructura semántica con header, main y footer.
  • Hero con llamada a la acción, sección de servicios o habilidades y un formulario de contacto funcional.
  • Estilos responsive, uso de Flexbox y un pequeño script (ej. validación o interacción de UI).

Entrega

Publica en GitHub Pages o Vercel y comparte el enlace. Incluye un README con instrucciones de instalación y vista previa.
